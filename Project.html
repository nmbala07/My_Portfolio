Project Summary:

The Expense Tracker with In-built AI is a smart web-based application designed to help users manage their personal or business finances with ease and intelligence. Built using Python, Django, and Node.js, this project not only records income and expenses but also provides AI-powered insights to help users make better financial decisions.

Features:

📊 User-friendly dashboard to track daily, weekly, and monthly expenses
💡 AI recommendations for saving money based on spending patterns
📈 Visual analytics like pie charts and bar graphs for spending categories
🧠 Natural language input for adding expenses (e.g., “Spent ₹500 on groceries”)
🔒 Secure authentication and multi-user support
📅 Recurring expense reminders and budget alerts
